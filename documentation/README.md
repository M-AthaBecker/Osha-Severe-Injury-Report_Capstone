# 📄 Documentation — OSHA Severe Injury Report Capstone

This folder contains the formal written deliverables for the OSHA Severe Injury Report Data Analytics Capstone Project.

---

## 📁 Contents

| File | Description |
|---|---|
| `OSHA_Capstone_Final_Writeup.docx` | Complete written report documenting all phases of the project including data cleaning, exploratory data analysis, SQL analysis, Python analysis, Tableau analysis, narrative, and conclusions |
| `OSHA_Capstone_Presentation.pptx` | 15-slide PowerPoint presentation designed for a 7-10 minute capstone presentation covering the full analysis from dataset overview through recommendations |

---

## 📝 Written Report

**File:** `OSHA_Capstone_Final_Writeup.docx`

The written report is organized into 11 sections:

| Section | Contents |
|---|---|
| 1. Dataset Overview | Source, scope, and what each row represents |
| 2. Data Dictionary | Definitions for all 25 columns plus NAICS reference |
| 3. Data Cleaning | All cleaning steps across Excel, SQL, and Python |
| 4. Exploratory Data Analysis | Dimensions, unique values, statistics, skewness, outliers |
| 5. Correlation Analysis | Hospitalization vs Amputation with correlation matrix |
| 6. Key Findings | Industry, geography, injury types, time trends |
| 7. SQL Analysis | NAICS join strategy, match status, employer standardization |
| 8. Python Analysis | Libraries, data cleaning, all 11 charts summarized |
| 9. Tableau Analysis | Data cleaning, all 12 worksheets, all 3 dashboards |
| 10. Narrative | Target audience, research questions, assumptions, limitations, recommendations |
| 11. Conclusion | Comprehensive summary of all findings |
